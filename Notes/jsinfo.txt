How to intigrate  JS with HTML Page?

1. Design HTML Forms Page.
2. Define javascript code  in <script> tag
3. Link that JS Code function to the related Tools(buttons, ....) for an execution.

HTML Form Tag :-  <form></form>

All the html form tags(textbox, radiobutton, checkbox, ...) must be enclosed within the <form> tag to get / set values. 

<form  name="name of form"  method="GET/POST"  action="pagename" ...>
	html coding
</form>
