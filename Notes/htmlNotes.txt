What is HTML?
	Full Form :-  HyperText Markup language

	html is used to create and design web pages. 

if we wants work with html, css and js, we don't need any specific software installation, just need browser. 
Coding can be written in any text editor(notepad, ....) but we use a third part tool called VSCode. 

Steps for HTML using VSCode:- 

1.  Open VSCode
2.  Create a new file then save that file with ".html" extension
3.  Code the program as per the requirement
4.  Execute program with browser.

-------------
html consist different types of commands which are called as tags.

every tag has the following syntax
		<tagname ....>Action</tagname>

As per html there are two types of tags
	1. Container Tags :-   it is having begining and ending tags
				<tagname ....>Action</tagname>
	2. Non-Container Tags :-  it is having only begining
				<tagname  ....... />
Structure :- 

	<html>
		<head>
			<title>title of the page</title>
			<meta .... />
			<style .... />
			<script .... />
		</head>
	
		<body>
			Actual web page code
		</body>
	</html>
